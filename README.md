# Telegram - бот на Python

## Описание

Telegram - бот с интерактивными упражнениями и системой отслеживания прогресса. Позволяет пользователям создавать персональные словари, практиковать переводы и проходить викторины.

## Архитектура
english-bot/
├── bot.py              # Главный запускаемый файл
├── bot_logic/          # Логика приложения
│   ├── database.py     # Работа с данными (слова, пользователи)
│   ├── states.py       # Состояния диалога (FSM)
│   ├── handlers.py     # Обработчики команд и сообщений
│   ├── keyboards.py    # Интерактивные клавиатуры
│   ├── practice.py     # Логика режима практики
│   └── quiz.py         # Логика викторины
├── data/               # Хранилище данных пользователей
│   └── words.json      # JSON файл со словами
├── Dockerfile          # Конфигурация Docker
├── docker-compose.yml  # Оркестрация контейнеров
├── requirements.txt    # Зависимости Python
└── .env               # Переменные окружения

## Функциональность

- Добавление новых для пользователя слов с переводом.
-  "Викторина" которая выдает пользователю слова в виде викторины
- Способность  бота по каманде /practice выдавать слова, а пользователю нужно написать перевод
- В случае неправильного ответа, бот может помочь по команде /help

# Собрать образ
docker build -t english-bot .

# Запустить
docker run -d \
  --name english-bot \
  -v $(pwd)/data:/app/data \
  --env-file .env \
  --restart unless-stopped \
  english-bot
